# data

Raw and derived research data should go here when experiments start.

Expected future artifacts:

- Sensitivity traces.
- Router training data.
- Evaluation metrics.
- Latency measurements.

Large model checkpoints should not be stored here.

## H1 categorized prompts

`h1_prompts.jsonl` is generated by `build_h1_prompts_jsonl.py`.

Each row has:

- `text`: prompt text consumed by `collect_h1_sensitivity.py --prompts_jsonl`.
- `category`: broad H1 query category.
- `source`: source benchmark.
- `source_id`: benchmark row id when available.

Current categories:

- `math`: `openai/gsm8k`
- `code`: `google-research-datasets/mbpp`
- `commonsense`: `Rowan/hellaswag`
- `science_reasoning`: `allenai/ai2_arc/ARC-Challenge`
- `retrieval`: `zai-org/LongBench/passage_retrieval_en`

Regenerate with:

```bash
python data/build_h1_prompts_jsonl.py --output data/h1_prompts.jsonl --per_category 50
```

Use in H1 with:

```bash
PROMPTS_JSONL=data/h1_prompts.jsonl bash experiments/h1-query-token-sensitivity/run_h1_full.sh
```
